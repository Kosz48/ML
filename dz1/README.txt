# Распознавание активности человека с использованием данных смартфонов

## Описание проекта

В этом проекте мы создаем модель машинного обучения для распознавания типа физической активности человека на основе данных, полученных с акселерометров и гироскопов смартфонов. Датасет содержит измерения от 30 добровольцев, выполняющих шесть видов активности:

1. Лежание
2. Сидение
3. Стояние
4. Ходьба
5. Спуск по лестнице
6. Подъем по лестнице

## Датасет

Мы используем датасет [Human Activity Recognition with Smartphones](https://www.kaggle.com/datasets/uciml/human-activity-recognition-with-smartphones) из Kaggle. Данные разделены на тренировочный и тестовый наборы, каждый из которых содержит признаки и метки активности.

## Цели проекта

- Загрузить и изучить данные.
- Провести анализ данных и вычислить основные характеристики.
- Обработать данные (пропущенные значения, категориальные признаки).
- Нормализовать данные.
- Обучить несколько моделей классификации (k-NN, случайный лес, SVM).
- Подобрать оптимальные гиперпараметры моделей.
- Оценить качество моделей и выбрать наилучшую.
- Выявить важные признаки и провести их анализ.

## Используемые методы и алгоритмы

- **Анализ данных**: визуализация, корреляционный анализ.
- **Предобработка данных**: масштабирование, кодирование меток.
- **Модели классификации**:
  - k-ближайших соседей (k-NN)
  - Случайный лес (Random Forest)
  - Метод опорных векторов (SVM)
- **Оценка моделей**: метрики точности, полноты, F1-меры, построение отчетов классификации.
- **Обработка дисбаланса классов**: использование параметра `class_weight`, применение SMOTE.
- **Снижение размерности**: удаление коррелированных признаков, метод главных компонент (PCA).

## Результаты

- **Случайный лес** показал наилучшую точность на валидационной выборке (~99%).
- Удаление сильно коррелированных признаков сократило количество признаков с 561 до 253 без существенной потери точности.
- Применение PCA позволило снизить размерность до 102 компонент при сохранении 95% дисперсии данных.

## Выводы

- Разработанная модель способна эффективно распознавать виды активности человека на основе данных с датчиков смартфона.
- Случайный лес является наиболее подходящей моделью для данной задачи.
- Сокращение размерности данных помогает улучшить эффективность модели без существенной потери качества.
